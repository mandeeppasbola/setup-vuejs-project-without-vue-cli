# setup-vuejs-project-without-vue-cli
Setting up latest Vue JS v3 (vue@next) project from scratch without using Vue CLI.

Including Single File Components (SFCs), Vue Router v4, SCSS
```sh
// Install dependencies
npm install
// To start
npm run start
// to build
npm run build
```
